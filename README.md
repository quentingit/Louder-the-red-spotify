### Getting started

- Install Node.JS if you don't have it - [Download page](https://nodejs.org/en/download)
- Install Expo if you don't have it - [Official documentation page](https://expo.io/learn)
    - `npm install -g expo-cli`
- Start Expo Project 
 - `expo start`
 or   `expo run:ios`
